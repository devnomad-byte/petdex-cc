## Hi Petdex team! 👋

We built **[petdex-cc](https://github.com/devnomad-byte/petdex-cc)** — a desktop pet companion that brings Petdex pets to **Claude Code** with deep integration.

### What it does

`petdex-cc` takes any Petdex pet and makes it **react to your coding in real-time**:

- **Real-time reactions** — pet changes animation when you read files, edit code, run commands, complete tasks, or hit errors
- **8-level progression** — Byte → Process → Thread → Module → Kernel → Neural → Quantum → Singularity, with escalating visual effects (glow, aura rings, floating particles, golden halo)
- **AI speech bubbles** — context-aware dialogue generated by Claude (or built-in preset lines)
- **Time-aware greetings** — morning, lunch, afternoon, evening, night messages
- **State persistence** — level and event count survive restarts (HMAC-signed)
- **Interactive** — drag, right-click menu, click easter egg, system tray

### Install in one command

```bash
npx petdex-cc install boba
```

That's it — the pet appears on your desktop and starts reacting to Claude Code immediately. All **1,569+ Petdex pets** are supported.
