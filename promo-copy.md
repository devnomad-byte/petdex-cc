## 6. 掘金
<!-- 发帖到掘金，技术博客风格，突出实现原理 -->

**标题:** 我给 Claude Code 做了个桌面宠物，它会实时响应你的编码操作

**正文:**
大家好，今天分享一个我做的开源项目 — petdex-cc。

## 这是什么？

petdex-cc 是一个 Claude Code 桌面宠物伴侣。安装后，一只萌宠会出现在你的桌面上，实时响应你的编码操作。

## 怎么实现的？

项目基于 Electron，通过 Claude Code 的 hooks 机制监听工具调用事件，将编码行为映射到宠物的状态变化：
- Tool 调用 → 宠物说话
- 任务完成 → 宠物庆祝
- 持续编码 → 经验值累积，等级提升

## 快速开始

```
npm install -g petdex-cc && petdex-cc install boba
```

1500+ 宠物可选，支持 Windows、macOS、Linux。

GitHub: https://github.com/devnomad-byte/petdex-cc
